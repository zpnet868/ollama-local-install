### 最低配置
CPU: Intel i5 或 AMD Ryzen 5
内存: 16GB
存储: 50GB SSD
GPU: 无（仅 CPU 运行）
### 推荐配置
CPU: Intel i7 或 AMD Ryzen 7
内存: 32GB
存储: 100GB NVMe SSD
GPU: NVIDIA RTX 3060 (12GB VRAM)
### 高性能配置
CPU: Intel i9 或 AMD Ryzen 9
内存: 64GB
存储: 500GB NVMe SSD
GPU: NVIDIA RTX 3090 (24GB VRAM) 或 A100 (40GB VRAM)
